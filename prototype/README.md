# Velvet Pong - Logic Prototype (No Visuals)

Pure game logic core for Velvet Pong.

This folder contains the JavaScript implementation of core mechanics, physics, RPG systems, and game loop based on the design documents in `/docs`.

**Focus**: Game code only. No rendering, no assets, no visuals.

## Current Status
- Turn 1: Core Pong Foundation (in progress)

See individual files for details and references to design docs.

Run with Node.js or in browser console for testing.
